
# 💬 Chat Application (CRUD)

## 📌 Description
A full-stack Chat Application built using Node.js, Express, MongoDB, and EJS.  
This project demonstrates complete CRUD operations (Create, Read, Update, Delete) with dynamic UI rendering.

---

## 🚀 Features
- ➕ Create new chats  
- 📖 View all chats  
- ✏️ Edit existing chats  
- ❌ Delete chats  

---

## 🛠️ Tech Stack
- Node.js  
- Express.js  
- MongoDB  
- EJS  
- Method-Override
